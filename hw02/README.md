# Проекты по Machine Learning

Этот репозиторий создан в учебных целях для выполнения домашних заданий и проектов по ML.

## 📌 О репозитории

Здесь собраны практические работы (домашки), выполненные в процессе изучения машинного обучения. Каждая папка содержит отдельный проект или домашнее задание с подробными комментариями и результатами.

## 📂 Структура

```
ml-course-homeworks/
  README.md                # (не обязательно) описание репозитория
  requirements.txt          # зависимости (можно обновлять по мере курса)
  .gitignore
  hw01_setup_tools/
    README.md               # это задание
    hw01_setup_tools.ipynb  # ноутбук с выполнением
  hw02_.../
    ...
```
## 🛠 Используемые технологии

- Python
- NumPy, Pandas
- Scikit-learn
- Matplotlib, Seaborn
- Google Colab, Jupiter
