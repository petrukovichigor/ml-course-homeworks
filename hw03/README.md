# Домашнее задание: Метод k-ближайших соседей (KNN)

Цель задания — самостоятельно исследовать метод k-ближайших соседей (KNN) на реальном датасете и сравнить его с несколькими базовыми моделями.

## Важное ограничение

Старайтесь выполнять задание преимущественно без использования GPT / Copilot для написания решения. Лучше использовать:
*   документацию библиотек,
*   материалы лекций,
*   собственные заметки,
*   встроенную документацию Python (`help()`).

Задача — пройти основные этапы небольшого ML-проекта и понять, когда KNN работает хорошо, а когда — хуже других моделей.

## 1. Выберите датасет

Возьмите один датасет из Kaggle для задачи классификации.

Желательно:
*   небольшой или средний размер,
*   в основном числовые признаки,
*   разумно применять расстояния между объектами.

### Подходящие варианты

*   **Iris Species** — Классический датасет.
*   **Wine / Wine Quality** — Химические признаки вина. Хорошо показывает влияние масштабирования и метрики расстояния.
*   **Breast Cancer Wisconsin** — Бинарная классификация с большим числом признаков. Удобно исследовать подбор гиперпараметров.
*   **Glass Classification** — Многоклассовая задача, где результат KNN сильно зависит от настроек.
*   **Dry Bean Dataset** — Более крупный и современный датасет с несколькими классами.
*   **Pima Indians Diabetes** — Классическая бинарная задача, удобная для экспериментов с предобработкой.
